# Travel Itinerary AI ✈️🗺️

**An AI-Powered Personalized Travel Planning Solution**

Travel Itinerary AI is an intelligent web application designed to eliminate travel planning stress. By leveraging the power of LLaMA 3.3 via the Groq API, it automatically generates personalized, day-by-day travel plans based on a user's destination, trip duration, and budget—reducing hours of tedious research down to seconds.

---

## 🌟 Features

*   **Personalized Itinerary Generation:** Tailored schedules based on user-defined inputs (destination, days, budget).
*   **Structured Day-Wise Schedules:** Clear breakdowns for **Morning, Afternoon, Evening, and Night** activities.
*   **Budget Estimation:** Smart financial planning with estimated costs for attractions and activities.
*   **AI-Powered Recommendations:** Highly curated attraction and dining recommendations using LLaMA 3.3.
*   **Dynamic Visuals:** Automated retrieval of high-quality destination images via Unsplash/Pexels APIs.
*   **Responsive UI:** A modern, clean, and mobile-friendly web interface.
